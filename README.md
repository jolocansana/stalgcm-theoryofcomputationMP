# stalgcm-theoryofcomputationMP
STAGLCM - Theory of Computation

## Token Class
### Attributes
* Content (string) - the actual string or symbol itself
* Type (integer) - classification of token
    * 0 - symbol
    * 1 - tag
    * 2 - attribute
    * 3 - attribute value
    * 4 - operator
    * 5 - value
